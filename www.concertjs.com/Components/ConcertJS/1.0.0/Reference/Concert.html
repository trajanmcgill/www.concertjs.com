<static> revertNameSpace()

Can be used to avoid namespace collision problems. Sets the global variable Concert back to what it was before this component assigned a new value to it. Usage: run the ConcertJS definition script (e.g., include the ConcertJS file via a script element on a web page), then immediately capture the object assigned to Concert in some other, non-conflicting variable for actual use, and then call Concert.revertNameSpace() to put back Concert to whatever value it had before.
