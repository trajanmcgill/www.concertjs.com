1) What does a simple Concert.js animation look like?

var sequence = new Concert.Sequence();
sequence.addTransformations(
  {
    target: document.getElementById("HelloDiv"),
    feature: "height",
    unit: "px",
    applicator: Concert.Applicators.Style,
    calculator: Concert.Calculators.Linear,
    easing: Concert.EasingFunctions.ConstantRate,
    keyframes: { times: [0, 1000], values: [0, 24] }
  });
sequence.begin();

Step 2

The most basic conceptual pieces involved in using Concert.js are the Transformation and the Sequence.

A Transformation is a single change applied over time to a single feature of a single object. For instance (as in this example), it could be a change in the height of a DOM element from 0px to 24px over 1000ms (1 second).

A Sequence is a series of changes applied to a collection of objects over time. It contains one or more transformations.

Anything we do with Concert.js will involve creating and working with a Sequence object.
